# Fruit Classification Using Deep Learning

An image-based fruit classification project developed using transfer learning with MobileNetV2.

## Features
- Image classification using deep learning
- Transfer learning with MobileNetV2
- Data augmentation
- Model training and evaluation
- Classification performance analysis

## Technologies
- Python
- TensorFlow
- MobileNetV2
- Keras
- NumPy
- Matplotlib

## Project Purpose
This project was developed as an academic deep learning project to classify fruit images using transfer learning techniques.

## Status
Completed academic project.
